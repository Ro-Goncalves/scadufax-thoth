# Rótulos de triagem

Este repositório usa os cinco papéis canônicos sem remapeamento. Em tracker local, essas strings devem ser usadas no campo Status: dos arquivos de issue.

| Rótulo canônico | String usada no repositório | Significado |
| --- | --- | --- |
| needs-triage | needs-triage | Um mantenedor precisa avaliar o issue. |
| needs-info | needs-info | Aguardando mais informações do reportante. |
| ready-for-agent | ready-for-agent | Totalmente especificado, pronto para um agente AFK. |
| ready-for-human | ready-for-human | Exige intervenção humana. |
| wontfix | wontfix | Não será corrigido. |

Quando uma skill mencionar um papel de triagem, use exatamente a string mapeada acima.
